Simulation

The simulation computes the cells' responses.

It runs in a separate thread to the main display GUI.  It reads the characteristics of the stimulus, then computes the excitation of each photoreceptor.

Next, all the cells' responses are computed based on the values of their inputs.

The response of the monitored cell is rendered visibily on the activity bar, and if it is a spiking cell, audibly through the speaker.
